# 🚀 Full Power Ubuntu 24.04 LTS RDP with OBS Studio

গুগল ক্লাউড শেলের (Google Cloud Shell) জন্য একটি সম্পূর্ণ অপ্টিমাইজড এবং ধাপে ধাপে সাজানো লিনাক্স আরডিপি (RDP) স্ক্রিপ্ট। এটি এক ক্লিকে সম্পূর্ণ **Ubuntu GNOME GUI**, **Google Chrome Browser**, এবং **OBS Studio** লেটেস্ট ভার্সনসহ ইনস্টল করে দেয়।

---

## ✨ বৈশিষ্ট্যসমূহ (Features)
* 🌟 **Ubuntu 24.04 LTS GNOME GUI:** উবুন্টুর সবচেয়ে লেটেস্ট এবং আকর্ষণীয় অফিসিয়াল ডেক্সটপ এনভায়রনমেন্ট।
* 🎥 **OBS Studio Pre-installed:** লাইভ স্ট্রিমিং বা স্ক্রিন রেকর্ডিংয়ের জন্য ওবিএস স্টুডিও আগে থেকেই রেডি করা।
* ⚙️ **রিসোর্স অপ্টিমাইজেশন:** ১৬ জিবি স্টোরেজ ও ৮ জিবি র‍্যামের সর্বোচ্চ স্পিড নিশ্চিত করতে ব্যাকগ্রাউন্ড মেমোরি বুস্টার যুক্ত।
* 📦 **সফটওয়্যার ম্যানেজার:** আরডিপির ভেতর যেকোনো অ্যাপ সহজে ইনস্টল/আনইনস্টল করার জন্য `Synaptic` এবং বেসিক প্যাকেজসমূহ অন্তর্ভুক্ত।
* 🛠️ **সিরিয়াল গাইড:** এলোমেলো এরর ছাড়া টার্মিনালেই ধাপে ধাপে নির্দেশনা ও ইনপুট সাজেশন।
