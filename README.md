# 🛠️ Google Cloud Shell Multi-RDP System (Ubuntu & Windows 10)

গুগল ক্লাউড শেলের (Google Cloud Shell) জন্য সম্পূর্ণ রেডিমেড আরডিপি স্ক্রিপ্ট কালেকশন। এখানে উবুন্টু ডেক্সটপ এবং উইন্ডোজ ১০ এর জন্য আলাদা আলাদা সেটআপ দেওয়া হলো। আপনার প্রয়োজন অনুযায়ী নির্দিষ্ট সেকশনটি ব্যবহার করুন।

---

## 🟥 সেকশন ১: Ubuntu 24.04 LTS RDP (With OBS Studio)
এটি এক ক্লিকে সম্পূর্ণ **Ubuntu GNOME GUI**, **Google Chrome Browser**, এবং **OBS Studio** লেটেস্ট ভার্সনসহ ইনস্টল করে দেয়।

### ✨ বৈশিষ্ট্যসমূহ (Features)
* 🌟 **Ubuntu 24.04 LTS GNOME GUI:** উবুন্টুর সবচেয়ে লেটেস্ট অফিসিয়াল ডেক্সটপ এনভায়রনমেন্ট।
* 🎥 **OBS Studio Pre-installed:** লাইভ স্ট্রিমিং ও স্ক্রিন রেকর্ডিংয়ের জন্য আগে থেকেই রেডি।
* ⚙️ **রিসোর্স অপ্টিমাইজেশন:** ১৬ জিবি স্টোরেজ ও ৮ জিবি র‍্যামের সর্বোচ্চ স্পিড বুস্টার যুক্ত।
* 📦 **সফটওয়্যার ম্যানেজার:** আরডিপির ভেতর অ্যাপ ইনস্টল/আনইনস্টল করার জন্য `Synaptic` টুলস অন্তর্ভুক্ত।
